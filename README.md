# Система обработки заявок

Веб-приложение для автоматизации обработки заявок пользователей.

Проект разработан на Python с использованием FastAPI, SQLAlchemy, SQLite и Docker.

---

# Возможности системы

## Пользователи

* регистрация и авторизация;
* роли: Пользователь / Администратор;
* профили пользователей;
* разграничение прав доступа.

## Работа с заявками

* создание заявок;
* изменение статусов;
* направления заявок;
* поиск и фильтрация;
* цветовая индикация статусов;
* комментарии и чат внутри заявки.

## Администрирование

* управление направлениями;
* просмотр пользователей;
* отчеты и аналитика;
* экспорт отчетов в Excel.

## Отчеты

* сводка по заявкам;
* группировка по статусам;
* группировка по направлениям;
* пользовательские отчеты;
* экспорт в Excel.

# Запуск проекта через Docker

## 1. Установить Docker Desktop

Скачать:
[https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

После установки проверить:

```bash
docker --version
docker compose version
```

---

# 2. Клонировать проект

```bash
git clone https://github.com/ivanovset500/project_practicum.git
```

Перейти в папку проекта:

```bash
cd project_practicum
```

---

# 3. Запустить проект

```bash
docker compose up --build
```

Docker автоматически:

* создаст контейнер;
* установит зависимости;
* запустит FastAPI;
* создаст SQLite-базу;
* откроет приложение на порту 8000.

---

# 4. Открыть проект

После запуска открыть:

```text
http://127.0.0.1:8000
```

---

# SQLite и Docker Volume

База данных SQLite хранится в папке:

```text
data/tickets.db
```

Используется Docker volume:

```yaml
volumes:
  - ./data:/app/data
```

Это позволяет:

* сохранять данные после пересборки контейнера;
* предотвращать потерю базы;
* хранить SQLite вне контейнера.

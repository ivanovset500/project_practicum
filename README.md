# Информационная система обработки заявок

Учебный MVP для проектного практикума: FastAPI + SQLite + HTML/CSS/JS.

## Возможности

- Регистрация и вход пользователей.
- Роли: пользователь, модератор, администратор.
- Создание заявок пользователями.
- Изменение статусов заявок: создана → в работе → выполнена.
- CRUD для заявок.
- Административная панель.
- Хранение данных в SQLite.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте в браузере:

```text
http://127.0.0.1:8000
```

## Тестовые пользователи

При первом запуске создается администратор:

- Логин: `admin`
- Пароль: `admin123`

## Структура проекта

```text
app/
  main.py          # маршруты FastAPI
  database.py      # подключение SQLite
  models.py        # модели БД
  auth.py          # авторизация и хеширование паролей
  templates/       # HTML-шаблоны
  static/          # CSS
```

## Обновление 1: комментарии, русские статусы и фильтры

Если проект уже запускался раньше и возникли ошибки из-за старой структуры базы, удалите файл SQLite-базы из папки проекта и запустите сервер заново. База создастся автоматически.
